# Teste-ecommerce

Intruções para baixar, executar e testar o projeto.

Baixar e instalar NPM em: 
	https://www.npmjs.com/get-npm

Baixando o projeto:

Utilizando o comando git clone no prompt de comando(cmd), copie a url e pressione enter. Exemplo:

git clone https://github.com/RicardoRibeiro21/sp-med-group-ui.git

Após clonar o repositório, precisamos adicionar as dependências do npm.

Basta abrir a pasta raíz do projeto, abrir o cmd e instalar o npm com o seguinte comando:

	npm install 

	Obs: Version npm - 6.13.7

O projeto além das dependências originais do npm, possuí outras necessárias para a funcionalidade e estrutura do projeto. Instal-as copiando os comandos:
	
	npm install react-bootstrap bootstrap
	npm install jquery popper.js
	npm install react-router-dom
	npm install react-loading

Com todas as dependências incluídas no projeto, podemos rodar o projeto para testá-lo. Utilize o cmd com o seguinte comando:
	
	npm start

	Obs: O projeto será executado na porta 3000 local.

Agora, podemos testar o projeto.
